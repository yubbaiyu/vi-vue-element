# 人工智能持续流水线平台

```bash
# clone the project
git clone 

# enter the project directory
cd xx

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
